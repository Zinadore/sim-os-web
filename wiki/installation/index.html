Installation

The first thing we need to do is to get Processing running in our machines.

Installing Processing

  • Download Processing 3.x from processing.org.

    Make sure to click on the link that matches your operating system. I have tested this project on Windows as well as the VM you have created for the class. Mac users can follow the instructions on the processing website, but Linux instructions should generally work.

  • Extract the archive. As of writing the latest version is Processing 3.5.3

    You can extract the archive to anywhere you have access. I'm using my Desktop. So the processing directory would be:

    • Windows: C:\Users\<username>\Desktop\processing-3.5.3
    • Linux: ~/Desktop/processing-3.5.3

  • Run Processing once to create some required folders.

    Run the executable inside the processing folder. For Windows it should be processing.exe and for Linux processing. Once it is done loading, you should be seeing something like this:

Installing SimOS

Our next step is to download the SimOS library and make available to processing. The following steps should help with that but if you get stuck or want extra information, you can always check the documentation here.

  • Download the zip found here.

  • Extract it to processing's library folder. The location of this folder will vary depending on your OS.

    • Windows: C:\Users\<username>\Documents\Processing\libraries
    • Debian VM: ~/sketchbooks/libraries

    The end result should look like:

    libraries/
|
|___ SimOS/
    |
    |___ examples/
    |
    |___ library/
    |   |
    |   |__SimOS.jar
    |
    |___ reference/
    |
    |___ library.properties

That should be it. To verify that you have installed it correctly, you can start processing and go to Sketch > Import Library and you should see SimOS at the end of the list, like this:
