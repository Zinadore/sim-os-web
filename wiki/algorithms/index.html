Algorithms

We are finally ready to implement our algorithms! SimOS requires you to provide implementations for the memory management algorithms and the process scheduling algorithms. The classes need to follow a pattern so SimOS can find them and load them. You can find a summary of that below.

Process schedulers

  • First Come, First Served : PS_FCFS
  • Shortest Job First : PS_SJF
  • Shortest Remaining Time Next : PS_SRTN
  • RoundRobin : PS_RR

Memory managers

  • First Fit : MS_FirstFit
  • NextFit : MS_NextFit
  • BestFit : MS_BestFit
  • WorstFit : MS_WorstFit

In the following sections we will implement First Come, First Served and First Fit. The rest will be up to you, according to your project description!

Adding more Java files

We need to add extra files to house our code. You can click the little downwards arrow next to your sketch and select New Tab, then give it the name of the class you want. So for PS_FCFS the file would be named PS_FCFS.java.
